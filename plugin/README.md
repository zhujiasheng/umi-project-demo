[![NPM downloads](http://img.shields.io/npm/dm/demo-plugin.svg?style=flat)](https://npmjs.org/package/demo-plugin)



## Install

```bash
# or yarn
$ npm install
```

```bash
$ npm run build --watch
$ npm run start
```

## Usage

Configure in `.umirc.js`,

```js
export default {
  plugins: [
    ['umi-plugin-demo-plugin', options],
  ],
}
```
